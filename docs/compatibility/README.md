# 🌍 兼容性深度解析

兼容性是 **otex** 的灵魂。为了让同一个 `.tex` 文件在不同的操作系统（Windows, Linux, macOS）、不同的语言环境（中文, 英文）以及不同的第三方模板中表现一致，otex 实现了一套复杂的自动化匹配逻辑。

## 🔍 核心内容

### [🖋️ 字体系统与跨平台配置](fonts.md)

- 详细解析 otex 如何加载字体。
- 提供 `windows`, `linux`, `adobe` 三套预设方案。
- **重点**：讲解如何使用 `fonts-dir` 选项实现“零安装”字体分发。
- **重点**：针对高级用户，提供三种手动绕过默认配置的方案。

### [💻 跨平台差异处理](cross-platform.md)

- 自动处理 Windows 与 Unix 系统的路径分隔符差异。
- 智能检测 Shell-escape 开启状态，适配 `minted` 等需要外部程序的包。
- 编译器适配建议 (XeLaTeX vs LuaLaTeX)。

### [🇨🇳 多语言自动本地化](localization.md)

- 解析 `english` 选项如何影响全局行为。
- 定理环境、算法环境、参考文献列表的自动翻译机制。
- `cleveref` 引用标签的中文适配（如：图 1-1, 表 2-2）。
