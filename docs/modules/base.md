# 🛠️ 基础工具模块 (`base`)

`base` 模块是 otex 加载的第一个功能模块，负责提供全包通用的基础工具和全局宏包。

---

## 1. 加载的宏包

- `pdftexcmds`: 提供跨平台的 shell-escape 检测支持。
- `xcolor`: 提供丰富的颜色支持（默认开启 `table` 选项）。
- `enumitem`: 提供高度可定制的列表环境（默认开启 `inline` 模式）。
- `siunitx`: 提供标准的科学计数法与单位排版。
- `microtype`: 提供微排版优化，减少过度伸缩。
- `indentfirst`: 确保章节后的首段也会正常缩进（符合中文排版习惯）。

## 2. 核心功能

- **颜色预定义**：定义了文档中常用的色彩空间。
- **环境初始化**：为后续模块（如 `code`）提供颜色和路径检测基础。

## 3. 配置选项

此模块为强制加载的基础设施，没有单独的开关选项。
