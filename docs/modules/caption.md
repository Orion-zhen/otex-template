# 🏷️ 图表标题模块 (`caption`)

`caption` 模块负责美化图片、表格的标题（Caption）样式，并处理其子标题（Subcaption）。

---

## 1. 加载的宏包

- `caption`: 用于定制标题的字体、间距和分隔符。
- `subcaption`: 智能加载，用于排版子图/子表。

## 2. 默认外观配置

otex 为你配置了既符合学术规范又略显精致的标题样式：

- **字体大小**：`small`。
- **标签加粗**：`bf` (如 **图 1**)。
- **分隔符**：冒号 (`labelsep=colon`)。
- **正文斜体**：`it` (仅针对西文字体生效)。

## 3. 智能冲突处理

在一些特殊类（如 `CjC` 计算机学报）或已经手动加载了 `subfig`/`subfigure` 的项目中，本模块会自动禁用 `subcaption` 的加载，以避免严重的内核冲突。
